1: в scss с помощью @extend можно расширять уже имеющиеся классы:
.dictaphone-size {
    width: 100px;
    height: 100px;
}

.dictaphone-section {
    @extend .dictaphone-size;
    border-radius: 50%;
    border: 1px solid black;
}
